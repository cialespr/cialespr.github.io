6 XHTML Rules

  • Elements must nest symmetrically.
  • Element names are case sensitive.
  • End tags are required
  • Empty elements are signified with a closing slash
  • Attribute values must be contained in quotation marks
  • Attribute-value pairs must be written in full (e.g. checked:checked;)

Definitions of HTML Vocabulary

Element
A pair of HTML tags containing content.
Attribute
Supported by some HTML elements, attributes provide more information about an element.
DTD
Document type definition(DTD) contains the elements, attributes and syntax rules for a languages like HTML and XML.
URL
Uniform Resource Locators(URL) are a communication protocol that allows hypertext linking.
W3C
Founded in 1994, at Massachusetts Institute of Technology (M.I.T.) the World Wide Web Consortium(W3C) sets standards for HTML and provides an open, nonproprietary forum for industry and academic representatives.
